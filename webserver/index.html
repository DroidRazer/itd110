Web Server Project

Vocabulary

Internet
A large web of servers that creates a large database of information any computer can access.
TCP/IP
Protocols that make up the internet's infrastructure.
LAN
Local Area Network. Connects devices within a limited area.
IP Address
Numerical label assigned to every device.
HTTP
Directs your browser how to retrieve the requested web page
Web server
Used to store, process, and retrieve web pages
Web Client
The web browser, or your end of the internet.
